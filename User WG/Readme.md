
### 工作组目标

- 基于不同的应用场景探讨对5G MEC的需求，明确EdgeGallery后续能力发展重点和方向；
- 确定TOP应用场景，明确基于EdgeGallery的验证Case和场景
### 工作组范围
- 工业现场、直播、视频、游戏等不同场景下对5G MEC的需求
- 社区优先集成的应用场景及意向应用厂商
