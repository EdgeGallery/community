# 人脸识别服务接口说明
## 功能介绍

上传照片进行人脸识别，返回待识别人脸所在的位置。

## 接口清单
### default
#### /upload
上传带有人像的图片
##### 请求语法

```
/upload
```

##### 请求参数

|参数 |位置 | 是否必选 | 类型 | 说明 |
|-----|-----|----|------|-----|
|file | formData |Y| file | 带有人像的图片 |

##### 返回数据

|返回码 |描述|
|-----|-----|
|200 | 上传成功 |
