## 项目目标
* 拉通各开发项目组，对需求进行详细方案设计，确保规划需求可以有效的落地
* 通过定期的技术方案讨论 和 网络安全研讨，打造安全可靠的EdgeGallery平台

## 项目范围
* 拉通Developer、App Store、User-mgmt的需求设计，保障版本的落地开发
* 遗留问题跟踪和闭环
* 提供技术方案求助通道，解决开发过程中遇到的问题
