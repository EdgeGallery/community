## 工作组目标
* 构筑EdgeGallery统一的云联接框架。实现云与边的协同。

## 工作组范围
* 实现EdgeGallery管理面与公有云集成，EdgeGallery能够与公有云进行应用市场联邦。
* 公有云能通过EdgeGallery实现公有云应用下沉边缘部署
* 实现应用边云协同，公有云应用下沉后可分享EdgeGallery网络能力，同时公有云可通过EdgeGallery开放公有云能力

## 工作组主要职责
* 讨论共有云和EdgeGallery的联合框架
* 讨论应用联邦集成方式
* 讨论公有云应用下沉部署流程
* 讨论应用边云协同方式，云联接器方案等。
